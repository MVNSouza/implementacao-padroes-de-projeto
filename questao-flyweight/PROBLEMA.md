# FLYWEIGHT
Uma agência vende ingressos para um grande evento onde são esperadas centenas de milhares
de pessoas no período de sua realização. Cada bilhete tem o `nome do evento`, o `local`, a `data`,
uma `cor` (ouro, prata ou bronze, que determina benefícios para o expectador) e uma
`identificação única` para cada comprador. Sabendo que você precisa ter um serviço eficiente
que utilize da melhor forma possível a memória dos computadores durante o uso do sistema
de vendas e gerenciamento dos ingressos, utilize o padrão FLYWEIGHT para utilizar de forma
eficiente a memória. Para impressão dos dados, informe quantos objetos foram criados em
memória após a venda de 100 mil bilhetes (Lembre de ter bilhetes com as três cores).
